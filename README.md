# 全A股云端逐股扫描器

本项目使用 GitHub Actions 免费云端 Runner，将全A股拆分为 20 个并行分片扫描，最后自动合并为 Excel。

## 输出文件

运行完成后会生成一个名为 `a-share-final-results` 的 Artifact，包含：

- `A股全市场逐股扫描结果.xlsx`
- `全A逐股结果.csv`
- `候选股票池.csv`
- `失败股票清单.csv`
- `覆盖率报告.json`

## 执行逻辑

1. 建立全A母集：
   - 沪深A股优先使用 BaoStock；
   - 北交所优先使用 AKShare；
   - 多接口失败时保留错误日志。
2. 将母集按代码顺序分成20片。
3. 20个GitHub云端任务并行抓取近5年历史行情。
4. 单只股票超过25秒自动跳过并记录。
5. 最后自动合并所有分片并生成Excel。

## 注意

- 这是程序化第一阶段扫描，不是最终投资建议。
- 数据失败股票不会自动视为淘汰。
- 最终候选需要进一步核验公告、财务、行业、估值、控制权、解禁减持和拥挤度。
